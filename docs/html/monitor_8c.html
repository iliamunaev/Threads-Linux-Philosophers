Detailed Description

Monitoring functions for the philosopher simulation.

This file contains functions responsible for monitoring philosopher status, including detecting deaths and checking if all philosophers have finished eating. The monitor thread ensures that the simulation ends when necessary.

Checks if a philosopher has died.

This function calculates the time since a philosopher's last meal and compares it with the allowed die time. If the philosopher has exceeded the time limit, the function marks the simulation as ended and prints a death message.

Thread safety:

  • Uses meal_mutex to safely access each philosopher's last meal time.
  • Uses end_mutex to update the simulation termination flag.
  • Uses print_mutex to print the death message without race conditions.
Parameters
envPointer to the environment structure.
iIndex of the philosopher to check.
Returns
1 if a philosopher has died, otherwise 0.

Monitor thread function.
