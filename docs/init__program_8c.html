Allocates and initializes the program environment.

This function dynamically allocates memory for the environment (t_env) and initializes all required parameters, including mutexes, philosopher structures, and simulation settings.

If memory allocation fails, the program prints an error message and immediately exits with EXIT_FAILURE. If environment initialization fails, it frees allocated resources before exiting.
